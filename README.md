# Command-Line To-Do List Application

A lightweight, terminal-based To-Do List application built in Python. This project demonstrates core programming fundamentals, local file persistence, and robust user input validation.

## 🚀 Features
* **Persistent Storage**: Saves tasks to a local text file so data is kept between sessions.
* **CRUD Operations**: Users can seamlessly create, read, and delete tasks.
* **Input Validation**: Safely handles empty inputs and invalid numeric selections without crashing.
* **Clean Interface**: Uses a simple, loop-driven command-line interface menu.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Core Modules:** `os` (for file checking)

## 📦 Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd todo-python-cli
   ```

2. **Run the application:**
   ```bash
   python todo.py
   ```
